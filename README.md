Generative AI used: non

Reason for not using AI: I have decided not to use generative AI for this project because I believe I understand concepts more deeply, learn faster, and retain information longer when I do not use AI for my programming projects. Programming languages are just like regular languages, and AI for programming is like Google Translate for languages. It is a great tool, but only if you already know the language or do not want to learn it. If you want to learn the language, you refrain from using a translator and instead try to actively pull the knowledge from your brain. I want to follow the same strategy for learning programming languages, algorithms, and methods.
